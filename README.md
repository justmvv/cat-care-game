# Кошкин день / A Cat's Day 🐈🐱

Обучающая игра (Flutter + Dart) для будущих владельцев кошек в стиле комикса.
Изометрическая комната, кошка Мурка, котёнок Барсик и владелец. Один игровой
день = 10 минут реального времени (08:00–22:00: утро → день → вечер → ночь).

## Геймплей

Кошки живут своей жизнью: хотят есть, играть, требуют внимания, ходят в лоток,
смотрят на птичек за окном, спят днём и устраивают вечерние «тыгыдыки».
Заскучав, они царапают диван и обои и скидывают вещи со стола — успейте
остановить (и предложите когтеточку!). Сюжетные события: птички за окном,
доставка коробки (кошачье счастье), потерянная мышка под диваном, вечерние
зумиос с риском для вазы.

Задания появляются справа с таймером. Выполняйте вовремя — иначе кошки
начинают ОРАТЬ, настроение падает. В конце дня — итог: 1–3 звезды или
поражение, плюс чему вы научились.

Управление: тап/клик по мискам (кормить), лотку (чистить), корзине с игрушками
(играть), кошке (гладить / останавливать шалость), когтеточке, двери, окну,
упавшим вещам (убирать).

## Звук

- Звуковые эффекты: включены по умолчанию, отключаются в настройках.
- Фоновая музыка: синтезированная рэгтайм-аранжировка по мотивам
  «Kitten on the Keys» (Zez Confrey, 1921, public domain).
  **По умолчанию выключена**, включается в настройках или на паузе.
- Все WAV-файлы сгенерированы скриптом `tools/gen_audio.py`
  (перегенерация: `python3 tools/gen_audio.py assets/audio`).

## Языки

Русский и English — переключение в настройках.

## Запуск

Требуется Flutter SDK 3.22+ (https://docs.flutter.dev/get-started/install).

```bash
cd cat_care_game
flutter pub get
flutter run            # выбрать устройство
flutter run -d chrome  # web
flutter run -d macos   # desktop (macOS)
```

## Структура

```
lib/
  main.dart                 — приложение, меню, настройки, справка
  core/strings.dart         — локализация RU/EN
  core/settings.dart        — настройки (persist через shared_preferences)
  core/sound_manager.dart   — SFX + музыка (audioplayers)
  game/models.dart          — модели: кошки, задания, комната
  game/game_controller.dart — симуляция дня: нужды, события, шалости, счёт
  render/iso.dart           — изометрическая математика
  render/scene_painter.dart — комикс-рендер комнаты и персонажей
  screens/game_screen.dart  — игровой экран, HUD, оверлеи
assets/audio/               — сгенерированные WAV
tools/gen_audio.py          — генератор звука/музыки
```
