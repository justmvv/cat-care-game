# A Cat's Day / Кошкин день 🐈🐱

An educational comic-style game (Flutter + Dart) for future cat owners.
A 2D side-view room, adult cat, kitten and their owner. One in-game day =
10 real minutes (08:00–22:00: morning → day → evening → night), with an
optional 2x fast-forward.

## Gameplay

The cats live their own lives: they want food, play and attention, use the
litter box, jump onto the sofa / table / windowsill, watch birds out the
window, nap in the afternoon and get the evening zoomies. When bored they
scratch the sofa and wallpaper, push things off the table, topple the floor
lamp, knock the flower pot off the windowsill and climb the curtains
(kitten only — and yes, curtains can be torn down).

Story events during the day: birds at the window, a parcel box delivery
(cat happiness), a fly hunt (chasing, leaping and swatting, hunting from on
top of the furniture), a lost mouse under the sofa, a thunderstorm with
lightning and scared cats hiding under the table, and a door that swings
open in a draught — close it in time or a cat escapes, the owner panics,
calls the pet rescue service, and the grumpy runaway comes home in a
carrier.

Tasks appear on the right with countdown timers. Complete them in time —
or the cats start YELLING and their mood drops. A too-dirty litter box ends
with a puddle right next to it. At the end of the day you get 0–3 stars
plus a list of cat-care lessons learned.

Controls: tap/click the bowls (feed), the litter box (scoop), the toy
basket (play with the teaser wand), a cat (pet it / stop mischief — a tap
anywhere near a naughty cat works), the scratching post, the door, the
window, fallen items (pick up), the toppled lamp (set upright). Tapping a
task card sends the owner to do the right thing automatically.

## Sound

- Sound effects: on by default, toggleable in Settings or via the HUD icon.
- Background music: a synthesized ragtime medley inspired by
  "Kitten on the Keys" (Zez Confrey, 1921) and "The Entertainer"
  (Scott Joplin, 1902) — both public domain. On by default, toggleable.
  On the web, browsers block autoplay: the music starts with the first tap.
- All WAV files are generated by `tools/gen_audio.py`
  (regenerate with: `python3 tools/gen_audio.py assets/audio`).

## Languages

Russian, English, Dutch and Spanish — switchable in Settings (with flags).
All UI strings live in `lib/core/strings.dart`.

## Running

Requires the Flutter SDK 3.22+ (https://docs.flutter.dev/get-started/install).

```bash
cd cat_care_game
flutter pub get
flutter run            # pick a device
flutter run -d chrome  # web
flutter build web      # PWA production build
```

The web build is PWA-ready: `web/index.html` contains a service-worker
auto-update script (skipWaiting + one reload), so every deploy reaches
players on their next visit. The build number shown at the bottom of the
main menu comes from `lib/core/version.dart` — bump it with every release.

## Project layout

```
lib/
  main.dart                 — app, menu, settings, help screens
  core/strings.dart         — RU/EN/NL/ES localization
  core/settings.dart        — settings (persisted via shared_preferences)
  core/sound_manager.dart   — SFX + looping music (audioplayers)
  core/version.dart         — build number shown in the menu
  game/models.dart          — models: cats, tasks, room layout
  game/game_controller.dart — day simulation: needs, events, mischief, score
  render/iso.dart           — 2D side-view projection math
  render/scene_painter.dart — comic renderer: room, cats, owner, weather
  screens/game_screen.dart  — game screen, HUD, overlays
assets/audio/               — generated WAVs
assets/icon/                — app icon (generated)
web/                        — favicon, PWA icons, manifest, index.html
tools/gen_audio.py          — sound/music generator
tools/gen_icons.py          — icon/favicon generator
```
